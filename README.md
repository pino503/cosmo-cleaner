# COSMO CLEANER

宇宙のゴミ掃除ロボがエイリアンと戦うアクションゲーム。

シングルHTMLファイル（外部依存なし）で動く、フロッピーディスク 1.44MB 制約のミニジャム向け作品です。

## 遊び方

GitHub Pages で公開しています。下記URLからすぐ遊べます：

**▶ Play: https://pino503.github.io/cosmo-cleaner/**

## 操作

| 操作 | 内容 |
|---|---|
| **ドラッグ** | プレイヤー移動 |
| **長押し** | 吸引（ゴミ・アイテムを吸い込む） |
| **FIRE ボタン** | 弾を発射 |

## 仕様

- 全 5 ステージ進行
- ボス戦あり
- 宝箱システム搭載
- 単一 HTML ファイル（`index.html`）に CSS/JS/アセット全部入り
- フロッピーディスク 1.44MB 制約のミニジャム向け作品
